date command in Linux

date command is used to display the system date and time. date command is also used to set date and time of the system. By default the date command displays the date in the time zone on which unix/linux operating system is configured.You must be the super-user (root) to change the date and time.

Syntax: 

date
day month date time format year

Other Functions of date

1: date :With no options, the date command displays the current date and time, including the abbreviated day name, abbreviated month name, day of the month, the time separated by colons, the time zone name, and the year.

Syntax :

date

Output :

Sun Sep 19 22:55:01 IST 2021

2: date -u :Displays the time in GMT(Greenwich Mean Time)/UTC(Coordinated Universal Time )time zone.

Syntax :

date -u

Output :

Sun Sep 19 06:11:31 UTC 2021

3: date %D :Display date as mm/dd/yy.

Syntax :

date "+%D"

Output :

9/19/2021

4: date %d: :Display the day of the month (01 to 31).

Syntax :

date "+%d"

Output :

09

Below is a small list of the some of the most common formatting characters.

  • %a - Locale’s abbreviated short weekday name (e.g., Mon)
  • %A - Locale’s abbreviated full weekday name (e.g., Monday)
  • %b - Locale’s abbreviated short month name (e.g., Jan)
  • %B - Locale’s abbreviated long month name (e.g., January)
  • %d - Day of month (e.g., 01)
  • %H - Hour (00..23)
  • %I - Hour (01..12)
  • %j - Day of year (001..366)
  • %m - Month (01..12)
  • %M - Minute (00..59)
  • %S - Second (00..60)
  • %u - Day of week (1..7)
  • %Y - Full year (e.g., 2019)

5: date %a: :Displays the abbreviated name for weekdays (Sun to Sat).

Syntax :

date "+%a"

Output :

Sun

6: date %A: :Displays full weekdays (Sunday to Saturday).

Syntax :

date "+%A"

Output :

Sunday

7: date %h or date %b: :Displays abbreviated month name (Jan to Dec).

Syntax :

date "+%h"
date "+%b"

Output :

Sep

8: date %B: :Displays full month name(January to December).

Syntax :

date "+%B"

Output :

September

9: date %m: :Displays the month of year (01 to 12).

Syntax :

date "+%m"

Output :

09

10: date %y: :Displays last two digits of the year(00 to 99).

Syntax :

date "+%y"

Output :

21

11: date %Y: :Display four-digit year.

Syntax :

date "+%Y"

Output :

2021
